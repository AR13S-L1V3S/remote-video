# Remote Video
Remote video utils with Javascript.
Capture video with video4linux from webcam, encode to webm with ffmpeg, stream to stream-m.



#### Features:
- capture
- webcam
- webm
- range streaming
- ffmpeg
- motion detection



## Operate
./start.sh
./stop.sh
./status
